Delaunay Triangulation

On this canvas you can visualize an animation for the algorithm that uses flipping edges to achieve Delaunay triangulation. In the first click, we identify the convex hull through the Graham scan algorithm. With the second click, a generic triangulation within the convex hull is generated from the set of points provided in the statement. A third click will initiate the animation that demonstrates the step-by-step process of the flipping algorithm to achieve a Delaunay triangulation.
