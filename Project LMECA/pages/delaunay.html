Delaunay

Method: Flip Edge

On this canvas you can visualize an animation for the algorithm that uses flipping edges to achieve Delaunay triangulation.

  • In the first click, we identify the convex hull through the Graham scan algorithm.
  • With the second click, a generic triangulation within the convex hull is generated from the set of points provided in the statement.
  • A third click will initiate the animation that demonstrates the step-by-step process of the flipping algorithm to achieve a Delaunay triangulation.

Delaunay

Method: Bowyer-Watson

On this canvas, you can visualize an animation of the Bowyer-Watson algorithm to achieve Delaunay triangulation. In the end, the super-triangle is erased, and the edges of the triangulation are corrected with the Graham scan algorithm.

  • In the first click, we identify the super-triangle.
  • With the second click, you can observe the animated algorithm.
